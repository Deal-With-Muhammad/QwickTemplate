# QWUIK Mobile POS

Expo (React Native) app powered by [HeroUI Native](https://heroui.com) +
uniwind + Tailwind CSS.

## Setup

```bash
npm install         # or yarn install
```

## Run

```bash
npm run ios         # iOS simulator
npm run android     # Android emulator
npm run web         # web preview
npm start           # interactive Expo dev server
```

EAS production build:

```bash
npm run build:ios
npm run submit:ios
```

## Where things came from

This project was originally cloned as the HeroUI Native library repo with the
app under `example/`. It's now a clean standalone Expo app — `heroui-native`
is pulled from npm (`^1.0.2`), the library source / monorepo workspace was
removed, and the app sits at the repo root.

If you ever want to upgrade HeroUI Native, just bump the version in
`package.json`:

```bash
npm install heroui-native@latest
```
